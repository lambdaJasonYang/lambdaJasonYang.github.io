# Linear or Cauchy pattern

```plantuml

@startuml
digraph G {
{rank=same; x; y;a;b}
{rank=same; f;g}
{rank=same; pp;tt;ee}
    a [label = "x"]
    b [label = "y"]
    g [label="*"]
    pp [label="f"]
    ee [label="f"]
    g -> pp;
    g -> ee;
    pp -> a;
    ee -> b;
    tt [label="*"]
    f -> tt
    tt -> x
    tt -> y
    
}

@enduml
```
