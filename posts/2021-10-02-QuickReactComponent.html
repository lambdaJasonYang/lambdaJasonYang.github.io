    Quick React Colocated Component library with Demo

    Posted on October 2, 2021
    Tags: prog, QuickCode, cloud, frontend

    1 Prelude

    The entire project is here

    https://github.com/lambdaJasonYang/colocatedReact

    I am sick and tired of all the tutorials online using bulky Storybook to hot-reload/demo their react components.
    Here we do not use any external tools to build our react library and hot-reload/demo in the SAME FOLDER.

    2 Summary

    Our general project name that contains both the component library and demo page is called colocatedReact

    This allows us to develop our react component library and demo it using the light weight default create-react-app in the SAME folder called colocatedReact.

    3 Components library (in the “componentlibrary” directory)

    mkdir colocatedReact
cd colocatedReact
mkdir componentlibrary
cd componentlibrary
mkdir common
mkdir react

    create a folder componentlibrary.
    Create subfolders for componentlibrary for your components dedicated to each framework (vue, react, common)

    3.1 componentlibrary

    3.1.1 componentlibrary/react

    cd react
touch index.js
touch package.json
mkdir src

    in colocatedReact/componentlibrary/core folder create index.js, package.json, src subfolder

    {
    "private": true,
    "name": "@mystuff/supercomponents",
    "version": "0.0.1",
    "main": "index.js",
    "scripts": {
      "test": "echo \"Error: no test specified\" && exit 1"
    },
    "author": "",
    "license": "ISC",
    "description": ""
  }
    export * from "./src";

    3.1.2 componentlibrary/react/src

    cd src
touch boo.js
touch index.js

    in src folder create file boo.js index.js

    export const SparkContext = 1231231;
    export {SparkContext} from "./boo"

    4 demo

    cd ~/colocatedReact
npx create-react-app demo

    this is where you will use the components library you created

    in your App.js

    import * as Spark from "@mystuff/supercomponents";
const { SparkContext } = Spark;

function App() {
  return (
    <div className="App">
        {SparkContext}
    </div>
  );
}

export default App;
    cd demo
npm install ~/colocatedReact/componentlibrary/react
npm start
