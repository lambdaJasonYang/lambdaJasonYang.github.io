# Open Delta neighborhoods

Neighborhood is a set N.

$$N(x_0,\delta) = \{x\in\mathbb{R}^n\ |\ \lvert x-x_0 \rvert < \delta \}$$

* 1-D neigborhood : Interval
* 2-D neighborhood: points in circle
* 3-D neighborhood: points in ball
