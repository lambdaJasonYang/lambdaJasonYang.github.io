    0.0.2 Setup code-server

    ~/.config/code-server/config.yaml

    CONVERT FROM THIS

    bind-addr: 127.0.0.1:8080
auth: password
password: idsjawij2382ruqfo
cert: false

    TO

    bind-addr: 0.0.0.0:8080
auth: password
password: root
cert: false

    In linux 127.0.0.1 means localhost but doesn’t allow access from LAN
    0.0.0.0 allows access from LAN

    run code-server instead of sudo code-server because the latter will assume root’s config.yaml.
