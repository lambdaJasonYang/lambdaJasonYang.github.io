    1 Quick Summary

    • A Go module is just the root directory containing go packages
      • go.mod go.sum similar to package.json package-lock.json
      • go package are declared INSIDE CODE of *.go file. eg: package duhr
        • package name are determined in golang code, not thru filename of *.go
        • go filename need not be the same as package name (but typically are)
          • bleh.go different filename from package duhr
          • Typically we use same filename and treat
            .go files synonymously with go packages
    • Go packages CAN ONLY IMPORT DIRECTORIES
      • import( "~/playGO/component" ) will import the Directory component
      • Go will auto-detect the package(s) package duhr from bleh.go inside the component directory.
    • Calling functions from other packages.
      Eg: package main in main.go calling functions from package duhr in bleh.go
      • Lowercase function name is package-public
      • Uppercase function name is package-private
      • Call exported function thru package name duhr.Somefunc()
        • Notice Somefunc() must be capitalized to be exported package-public
        • duhr.somefunc() WILL FAIL

    2 Go module

    2.1 go.mod: Init root dir

    mkdir projectFolder
cd projectFolder
go mod init github.com/UserJY/playGO

    This will autogenerate go.mod file

    • github.com/UserJY/playGO is the name and the root directory of our module

    2.2 go.sum: Download dependencies

    go get -u github.com/gin-gonic/gin

    This will autogenerate go.sum file

    2.3 Explore go.mod go.sum

    3 Packages

    3.1 main.go component/bleh.go

    package main

import (
	"fmt"
	"github.com/UserJY/playGO/component" 
  //we import the DIRECTORY "component"
  //we do not import bleh.go or duhr
)

func main(){
	fmt.Println(duhr.Somefunc())

}

    mkdir component Create component directory

    package duhr

func Somefunc() int { // "S" in Somefunc() MUST BE CAPITALIZED
	return 2
}

    “S” in Somefunc() MUST BE CAPITALIZED since it is package-public
    This allows main.go to use Somefunc()
    duhr.somefunc() WILL FAIL since it is lowercase package-private

    4 Running

    try below (will fail because we are running locally)

    go run .

# main.go:5:2: no required module provides package github.com/UserJY/playGO/bleh; to add it:
#         go get github.com/UserJY/playGO/bleh

    To run locally:

    go mod edit -replace github.com/UserJY/playGO/component=~/playGO/component

    this will add replace github.com/UserJY/playGO/component => ~/playGO/component to your go.mod file
